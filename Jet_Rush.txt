Jet RushBoost through the skies and escape the islandGithub Link to Code: https://github.com/abbydups/Jet-Rush-2D-Game
Changes Implemented from the Design Document1. Scenery changes when a milestone is reached was removed from the game core loop. The reason for this is that it cluttered the background which causes the obstacles that the player must avoid hard to be seen. I found that the game benefitted more from a plain sky and ocean water background so that the player can focus on seeing the obstacles.2. The hoverboard was replaced with Jetpack because I could not find any art assets that matches the jet powered style hoverboard, the closest one I could find was a jetpack.3. Collectible treasures were decided to just be coin instead of diamonds and the score was determined by the distance travelled by the player instead of the number of coins that was collected. 4. The Escape control was removed and replaced with a button instead, so that I can incorporate sounds and animations on it. It makes the game feels more interactive and fun. 5. Instead of six obstacle types, I decided to only make it four, which is the tree, drone, blimp, and bird. This made the game less challenging and more enjoyable. This decision was done through multiple play testing and letting friends play the game. They said that they did not like having too many obstacles because it created too much friction in the game.
